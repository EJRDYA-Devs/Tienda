# **Evaluacion de conocimientos**
-	Se crearon las migraciones con sus respectivos modelos y Factory conforme lo estipulado.
-	Se uso un témplate personalizado para generar las interfaces graficas.
-	Se uso node para instalar y compilar los scripts y css.
-	Para compilar todos los estilos se debe ejecutar npm run dev o prod.
-	Para los procesos de los CRUD y los reportes se usó LIVEWIRE como Framework Full Stack de Laravel.
-	Ambas vistas, tanto la de productos como transacciones tienen su boto para exportar.
